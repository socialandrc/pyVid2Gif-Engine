# Contributing to pyShortVid2Gif-Engine

Thanks for your interest in contributing!

## Getting Started

1. Fork and clone the repo.
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Install dev dependencies: `pip install -r dev-requirements.txt`
4. Run linting and tests:
   ```bash
   python -m ruff check . --fix
   python -m pytest tests/ -v
   ```
5. Commit with a clear message and push.
6. Open a pull request.

## Guidelines

- Keep commits atomic and well-messaged.
- Add tests for new features or fixes.
- Run linting before submitting.
- Update README if you change user-facing behavior.

## Reporting Bugs

Use GitHub Issues with a minimal reproduction and your environment (OS, Python version, MoviePy version).
